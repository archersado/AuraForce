# System Design - 系统架构设计

本目录包含 AuraForce 项目的技术架构设计文档。

## 🏗️ 系统架构

### 整体架构
```
┌─────────────────────────────────────┐
│         Frontend Layer              │
│  Next.js 16 + React 18 + TypeScript │
└───────────┬─────────────────────────┘
            │
            │ (Next.js API Routes)
┌───────────┴─────────────────────────┐
│         Backend Layer               │
│  Node.js + NextAuth 5 + MCP SDK    │
└───────────┬─────────────────────────┘
            │
            │ (Prisma ORM)
┌───────────┴─────────────────────────┐
│      Database Layer                 │
│   MySQL/MariaDB via Prisma         │
└─────────────────────────────────────┘
```

### 技术选型

#### 前层
- **Next.js 16**: React 框架，支持 SSR、SSG
- **React 18**: UI 库
- **TypeScript**: 类型安全
- **TailwindCSS**: 样式框架
- **Radix UI**: UI 组件库
- **Cherry Markdown**: Markdown 编辑器

#### 后端
- **Node.js**: 运行时
- **Next.js API Routes**: API 框架
- **NextAuth 5**: 身份认证
- **MCP SDK**: Model Context Protocol

#### 数据库
- **MySQL/MariaDB**: 关系型数据库
- **Prisma ORM**: ORM 框架
