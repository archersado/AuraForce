# Cherry Markdown 简介

## 什么是 Cherry Markdown

Cherry Markdown 是由腾讯开源的一款所见即所得（WYSIWYG）的 Markdown 编辑器，具有以下特点：

- 🎨 所见即所得编辑
- 📝 完整支持 Markdown 语法
- 🔌 丰富的插件生态
- 🚀 高性能
- 📱 移动端友好
- 🌍 支持中英文

## 与 Tiptap 对比

| 特性 | Cherry Markdown | Tiptap |
|------|----------------|--------|
| 类型 | 独立编辑器 | 底层编辑器框架 |
| 开箱即用功能 | ✅ 丰富 | ⚠️ 需配置扩展 |
| 中文支持 | ✅ 原生支持 | ✅ 良好 |
| 性能 | ⚡ 优秀 | ⚡ 快速 |
| 定制性 | 🟡 中等 | 🚀 非常高 |
| 插件生态 | 🔌 内置丰富 | 🧩 需手动安装 |

## 官方资源

- GitHub: https://github.com/Tencent/cherry-markdown
- 官网: https://cherry-md.dev/
- React 集成: 可使用 `cherry-markdown` 和 React 包装器

## 核心功能

- 标题、列表、引用、代码块
- 表格支持
- 图片上传
- 数学公式（KaTeX）
- 脑图、流程图
- 导出为 HTML/图片
- 快捷键支持
