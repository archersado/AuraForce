# Record - 过程信息文档

本目录包含 AuraForce 项目开发过程中与 User（用户）交互产生的所有过程信息文档。

---

## 📋 目录说明

本目录不包含工作模式文档或项目管理文档，仅包含：

1. **过程文档** - 记录开发过程中的操作、决策和结果
2. **同步报告** - 记录 Linear 同步、MCP 配置等的完成情况
3. **归档文档** - 旧版本或过时的文档的归档
4. **分析报告** - 项目分析、研究分析等报告

---

## 📝 文档说明

### 归档策略

- 过时的文档会定期归档到 `ARCHIVE.md` 或创建新的归档文件
- 归档文件命名格式: `ARCHIVE_YYYY-MM-DD.md`
- 重要的过程文档会保留在此目录，仅删除重复或无用的文档

### 文档更新

- 过程文档通常只读，记录历史过程
- 如需更新相关内容，请更新到对应的工作模式文档或项目管理文档
