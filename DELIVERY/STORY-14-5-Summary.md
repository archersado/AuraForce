## 技术实现

- **PPTX 解析**: JSZip (解析 ZIP 格式的 PPTX 文件)

### 关键功能实现细节

#### 1. PPTX 文件解析
- PPTX 本质是 ZIP 压缩包
- 解析 `ppt/slides/slide{N}.xml` 获取幻灯片内容
- 通过 `ppt/slides/_rels/slide{N}.xml.rels` 获取图片引用
- 从 `docProps/core.xml` 提取演示文稿标题

#### 2. 图片提取
- 遍历关系文件找到图片引用
- 从 `ppt/media/` 或 `ppt/slides/media/` 加载图片
- 创建 Blob URL 用于前端显示
- 组件卸载时自动清理 Object URLs

#### 3. 导航系统
- **按钮导航**: 上一页、下一页按钮
- **键盘导航**: ← → Space Home End Esc
- **缩略图导航**: 点击缩略图直接跳转
- **自动轮播**: 3 秒间隔，可暂停

## 遇到的挑战与解决方案

### 挑战 1: PPTX 文件格式
**问题**: PPTX 是复杂的 XML 结构，不易解析
**解决**: 使用 JSZip 解压，然后使用 DOM Parser 解析 XML，简化处理

### 挑战 2: 图片提取
**问题**: 图片分散在不同目录，需要通过关系文件查找
**解决**: 解析 `slide{n}.xml.rels` 文件，找到图片的相对路径，然后加载

### 挑战 3: 内存管理
**问题**: 创建大量 Object URL 可能导致内存泄漏
**解决**: 使用 useRef 存储引用，在组件卸载时统一清理

### 挑战 4: TypeScript 类型问题
**问题**: JSZip 导入方式与 TypeScript 配置不兼容
**解决**: 使用 `import * as JSZip` 方式导入，兼容性更好

## 已知限制

### 功能限制
1. 仅支持 .pptx 格式（Office 2007+）
2. 不支持 .ppt 格式（旧版 PowerPoint）
3. 不完全还原样式和布局
4. 不支持动画、视频、音频

### 性能限制
1. 大型文件（>50MB）加载较慢
2. 图片多的演示文稿性能下降

### 兼容性限制
1. 某些复杂图表无法预览
2. SmartArt 图形不支持
3. 嵌入式字体不支持
