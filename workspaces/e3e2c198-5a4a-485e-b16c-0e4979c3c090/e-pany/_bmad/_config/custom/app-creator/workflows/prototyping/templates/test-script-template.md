## Introduction Script
*(Read to participant at start)*

```
欢迎！感谢你今天抽出时间参与这次测试。

我是 [Name]，是 [产品] 的设计师。

今天，我们将测试一个产品的早期版本原型。

**重要提醒：**

1. 我们在测试设计，不是测试你
2. 不存在"正确"或"错误"答案
3. 你的诚实反馈对我们最有价值
4. 随时可以提问或暂停

整个过程大约45分钟。准备好了吗？
```

## Pre-Test Questions

```
1. 在日常生活中，你使用过[相关产品类别]的产品吗？
   - 使用过哪些？
   - 使用频率如何？

2. 你希望这样的产品解决什么问题？

3. 你对这类产品的期望是什么？
```

## Post-Test Questions

```
1. 整体体验如何？

2. 最喜欢哪些方面？
   - [Aspect 1]
   - [Aspect 2]

3. 最不喜欢哪些方面？
   - [Aspect 1]
   - [Aspect 2]

4. 哪里让你感到困惑或困难？

5. 你会把这个产品推荐给朋友吗？为什么？

6. 有什么功能或改进建议？

7. 还有什么想告诉我们的吗？
```
