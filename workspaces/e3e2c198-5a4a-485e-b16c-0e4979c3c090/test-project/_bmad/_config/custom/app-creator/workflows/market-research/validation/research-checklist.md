# 市场研究检查清单

## Step 2: 研究范围定义
### 研究目标
- [ ] 明确主要研究目标
- [ ] 识别关键研究问题
- [ ] 定义研究用途
### 目标市场
- [ ] 定义地理范围

- [ ] 识别细分市场
### 时间范围

### 研究深度
- [ ] 确定研究深度级别
- [ ] 选择重点分析维度
- [ ] 确定竞品分析数量

### 直接竞争对手
- [ ] 识别3-5个主要直接竞品

- [ ] 分析竞品相似度

- [ ] 识别替代解决方案

- [ ] 评估替代威胁
### 竞争格局
- [ ] 评估市场竞争程度
- [ ] 识别红海/蓝海特征
- [ ] 分析市场结构
### 优先级排序
- [ ] 评估竞争威胁
- [ ] 选择学习对象
- [ ] 确定深度分析对象

### 产品分析

- [ ] 每个竞品的威胁
