# Development Tasks - 开发任务

本目录包含 AuraForce 项目的技术开发任务文档。

## 📋 任务类型

### 前端任务
UI 组件开发、前端逻辑实现、页面开发等。

### 后端任务
API 开发、业务逻辑实现、数据库操作等。

### 测试任务
单元测试、E2E 测试、集成测试等。

### 部署任务
部署配置、CI/CD、环境搭建等。

## 🎯 任务创建

### Task 模板

参考 [Product Management > Tasks > Task 模板](../../pm/tasks/README.md#task-模板)

### 创建任务流程

1. 从 Story 拆解 Task
2. 明确任务类型和优先级
3. 估算工作量
4. 分配负责人
5. 创建任务文档
