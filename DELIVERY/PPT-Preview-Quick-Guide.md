# PPTX 预览功能 - 快速使用指南

## 功能概述

PPTX 预览功能允许用户在工作区直接查看 PowerPoint 演示文稿，无需下载或安装额外软件。

## 支持的文件格式

- ✅ `.pptx` - PowerPoint Open XML (Office 2007+)
- ❌ `.ppt` - 旧版 PowerPoint（建议转换为 .pptx）

## 如何使用

### 1. 上传 PPTX 文件

1. 点击工具栏的"Upload"按钮
2. 选择一个或多个 `.pptx` 文件
3. 文件会自动上传到工作区

### 2. 查看演示文稿

1. 在文件树中双击 `.pptx` 文件
2. 文件标签页会在编辑器中打开
3. 自动加载并显示第一张幻灯片

### 3. 导航控制

#### 按钮导航
- **← 上一页**: 跳转到上一张幻灯片
- **→ 下一页**: 跳转到下一张幻灯片
- **播放/暂停**: 自动轮播幻灯片

#### 键盘快捷键

| 键 | 功能 |
|---|---|
| `←` | 上一页 |
| `→` | 下一页 |
| `空格` | 下一页 |
| `Home` | 第一页 |
| `End` | 最后一页 |
| `Esc` | 退出全屏 / 停止播放 |

#### 缩略图导航
- 底部显示所有幻灯片缩略图
- 点击任意缩略图直接跳转
- 当前幻灯片会高亮显示

### 4. 全屏模式

1. 点击工具栏的"全屏"按钮最大化预览
2. 全屏模式下：
   - 工具栏隐藏
   - 右上角显示"退出全屏"按钮
   - 所有键盘快捷键仍然可用
3. 按 `Esc` 键或点击"退出"按钮退出全屏

### 5. 下载文件

- 点击"下载"按钮保存原始 PPTX 文件到本地

## 常见问题

### Q: 为什么我的 .ppt 文件无法预览？
A: 当前仅支持 `.pptx` 格式（Office 2007+）。请使用 PowerPoint 将文件另存为 `.pptx` 格式。

### Q: 为什么某些幻灯片看起来不一样？
A: PPTX 预览显示基本内容和结构，不会完全还原：
- 复杂的样式和布局
- 动画效果
- SmartArt 图形
- 视频/音频内容

### Q: 如何提高加载速度？
A: 大文件（>50MB）或包含大量图片的演示文稿可能加载较慢。建议：
- 优化图片大小
- 压缩演示文稿
- 使用本地文件而非从网络加载

### Q: 可以保存或编辑演示文稿吗？
A: 当前版本仅支持预览，暂不支持编辑。即将推出的版本将支持基本编辑功能。

### Q: 可以打印演示文稿吗？
A: 当前版本不支持打印。请下载文件后使用 PowerPoint 打印。

### Q: 支持协作功能吗？
A: 暂不支持多人同时查看同一演示文稿。未来版本将加入协作功能。

## 技术支持

如遇问题，请：

1. 检查浏览器控制台是否有错误信息
2. 确认文件格式为 `.pptx`
3. 尝试下载文件并在本地打开验证文件完整性
4. 联系技术支持并提供：
   - 错误截图
   - 文件大小和复杂度
   - 浏览器信息
